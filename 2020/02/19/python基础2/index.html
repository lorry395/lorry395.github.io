    三、文件操作

    1、初识⽂件操作

    使⽤open()函数来打开⼀个⽂件：open（文件名（路径），mode=“打开文件的方式”，encoding=“字符集utf-8”）

    打开⽂件的⽅式: r, w, a, r+, w+, a+, rb, wb, ab, r+b, w+b, a+b 默认使⽤的是r(只读)模式。

    rb, wb, ab, 用来读取非⽂本⽂件的时候，比如读取MP3、图像、视频等信息的时候就需要⽤到，因为这种数据是没办法直接显⽰出来的，⽂件上传下载的时候还会⽤到，以及直播， 实际上都是这种数据。

    2、绝对路径和相对路径

    绝对路径:从磁盘根⽬录开始⼀直到⽂件名。

    相对路径:同⼀个⽂件夹下的⽂件，相对于当前这个程序所在的⽂件夹⽽⾔，如果在同⼀个⽂件夹中，则相对路径就是这个⽂件名，如果在上⼀层⽂件夹。则要../ 。

    3、只读(r, rb)

    只读r

    
f = open("文件路径.txt",mode="r", encoding="utf-8") # 打开文件 
content = f.read() # 读取内容,文件大时不要用 
print(content) # 打印读取的内容 
f.close() # 关闭文件

    需要注意encoding表⽰编码集,根据⽂件的实际保存编码进⾏获取数据,对于我们⽽⾔,更多的是utf-8。

    rb. 读取出来的数据是bytes类型, 在rb模式下,不能选择encoding字符集。

    
f = open("文件名.txt",mode="rb" )
content = f.read()
print(content)
f.close()

    rb的作⽤: 在读取非⽂本⽂件的时候。比如读取MP3、图像、视频等信息的时候就需要⽤到。

    4、只写(w, wb)

    w.写的时候注意如果没有⽂件则会创建⽂件,如果⽂件存在,则将原件中原来的内容删除,再写入新内容。

    
f = open("文件", mode="w", encoding="utf-8")
f.write("oeasys.cn")
f.flush() # 刷新
f.close()

    wb模式下,可以不指定打开⽂件的编码,但是在写⽂件的时候必须将字符串转化成utf-8

    
f = open("文件", mode="wb")
f.write("oeasys.cn".encode("utf-8"))
f.flush()
f.close()

    5、追加(a, ab)

    a,在追加模式下,我们写入的内容会追加在⽂件的结尾。

    
f = open("文件", mode="a", encoding="utf-8")
f.write("oeasys.cn")
f.flush()
f.close()

    6、r+读写

    对于读写模式,必须是先读,因为默认光标是在开头的，准备读取，当读完了之后再进⾏写入。我们以后使⽤频率最⾼的模式就是r+。

    
f = open("文件", mode="r+", encoding="utf-8")
content = f.read()
f.write("oeasys.cn")
print(content)
f.flush()
f.close()
结果:
#正常的读取之后, 写在结尾

    7、w+写读

    先将所有的内容清空,然后写入,最后读取。

    
f = open("文件", mode="w+", encoding="utf-8")
f.write("oeasys.cn")
content = f.read()
print(content)
f.flush()
f.close()

    8、a+写读(追加写读)

    a+模式下,不论先读还是后读,都是读取不到数据的。

    
f = open("文件", mode="a+", encoding="utf-8")
f.write("oeasys.cn")
content = f.read()
print(content)
f.flush()
f.close()

    9、读取⽂件的⽅法

    read()将⽂件中的内容全部读取出来。弊端: 占内存，如果⽂件过⼤，容易导致内存崩溃。

    
f = open("文件路径", mode="r", encoding="utf-8")
content = f.read()
print(content)

    readline()⼀次读取⼀⾏数据。

    
f = open("文件路径", mode="r", encoding="utf-8")
content = f.readline()
content2 = f.readline()
content3 = f.readline()
content4 = f.readline()
content5 = f.readline()

    循环读取,这种⽅式是组好的,每次读取⼀⾏内容,不会产⽣内存溢出的问题。

    
f = open("文件", mode="r", encoding="utf-8")
for line in f:
print(line.strip())

    注意: 读取完的⽂件句柄⼀定要关闭 f.close()。

    10、其他操作⽅法

    (1)seek(n) 光标移动到n位置, 注意, 移动的单位是byte。所以如果是UTF-8的中⽂部分要是3的倍数。

    通常我们使⽤seek都是移动到开头或者结尾。

    seek(偏移量，位置)

    移动到开头: seek(0)

    移动到结尾: seek(0,2)

    seek的第⼆个参数表⽰的是从哪个位置进⾏偏移, 默认是0, 表⽰开头, 1表⽰当前位置, 2表⽰结尾。

    (2)tell()获取到当前光标位置

    print(f.tell())

    (3)truncate() 截断⽂件

    从开头截断到光标位置，删掉光标后⾯的所有内容,如果给参数,从开头截断到参数位置。

    在r+模式下,如果读取了内容,不论读取内容多少,光标显⽰的是多少,再写入或者操作⽂件的时候都是在结尾进⾏的操作。

    所以如果想做截断操作,记住了,要先挪动光标,挪动到你想要截断的位置,然后再进⾏截断。
